Think fast!

This is a webpage used to test your reaction speed on things. 

For now, it only supports single digit numbers. This could be useful if you want to practice pressing numbers on the numpad, useful for when your job is filling numbers in an excel sheet. 

Try now at [afiqket.github.io/thinkfast](https://afiqket.github.io/thinkfast/)!
